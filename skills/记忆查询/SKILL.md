# 记忆查询

## 描述

从飞书多维表格记忆库中检索历史记忆，使用本地向量搜索找到语义最相关的内容。

## 触发时机

当用户需要从历史中查找信息时触发：
- "我之前记了什么..."、"帮我找找关于...的记忆"
- "历史上有没有...相关的内容"
- "你还记得我说过...吗"
- "之前有没有解决过...这个问题"

## 操作步骤

1. 从用户的问题中提取 2-3 个关键词或短语（捕捉语义核心）
2. 调用向量搜索：

```bash
node /path/to/mem-feishu/dist/index.js search \
  --query "<关键词1 关键词2 关键词3>" \
  --limit 10
```

3. 解析返回的 JSON，过滤 `score > 0.3` 的结果
4. 如果有相关结果：
   - 整理成可读的摘要回复用户
   - 注明每条记忆的标签、来源、大致时间
5. 如果没有相关结果：
   - 改用 `recent --limit 10 --format` 返回最近记忆
   - 告知用户未找到强相关内容，展示最近记录供参考

## 注意

- 搜索结果 score 越高代表越相关（0~1 范围）
- 整理结果时要简洁，不要原文照搬，要归纳总结
