# 记忆保存

## 描述

将用户希望保留的信息永久保存到飞书多维表格记忆库，支持自动提取关键内容并分配分类标签。

## 触发时机

当用户说出以下意图时触发：
- "记住..."、"帮我记住..."、"保存这个..."
- "记录一下..."、"存档..."、"注意..."
- "以后要记得..."、"这个很重要，记住"

## 操作步骤

1. 从用户的话中提取核心信息（去掉客套话，保留实质内容）
2. 根据内容分配 1-3 个中文标签，从以下候选中选择最合适的：
   - 决策、配置、调试、团队、偏好、架构、计划、问题、解决方案、工具、手动
3. 识别当前项目名称（当前工作目录的目录名）
4. 调用以下命令保存记忆：

```bash
node /path/to/mem-feishu/dist/index.js save \
  --content "<提取的核心内容>" \
  --tags "<标签1>,<标签2>" \
  --source "openclaw" \
  --project "<项目名>"
```

5. 确认保存成功后，简洁地告知用户：「已保存到飞书记忆库 ✓」

## 注意

- 提取的内容要简洁、信息量高，去掉冗余表达
- 不要问用户是否确认，直接保存后告知结果
- 如果命令返回错误，告知用户并检查环境变量是否配置正确
